# Future_in_dcsa_T1
Sales Performance Analysis

In this task we look at superstore sales dataset to perform a business sales analysis
We identify revenue trends and business growth opportunties. The goal of this analysis is to understand which products, categories and regions generate the most sales and give reccomendations on how to improve performance.
To get started i used Visual studio code(python) to analyze the data and the trends, therefore i created a pdf report of the analysis by talking about the analyzed data and giving recommendations on how to improve performance.
